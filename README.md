# vue-communique

Advanced Vue notification system. Alpha release, not production ready.

## Project setup

```sh
yarn
```

### Compiles and hot-reloads for development

```sh
yarn run serve
```

### Compiles and minifies for production

```sh
yarn run build
```

### Lints and fixes files

```sh
yarn run lint
```

### Run your unit tests

```sh
yarn run test:unit
```
